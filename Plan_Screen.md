# Screen: Plan

## Purpose

This screen displays the user's diet plan targets for the current day and potentially the week, calculated locally according to the ZigZag method.

## Content Displayed

* **Daily Calorie Target:** The calculated maximum calories for the current day.
* **Daily Macronutrient Targets:** Calculated targets for Protein, Carbohydrates, and Fat (in grams) for the current day.
* **Day Type Indicator:** Clearly indicates if the current day is planned as a "Workout Day" (higher calories) or "Rest Day" (lower calories).
* **Weekly Overview (Optional):** May show targets for other days in the week.

## Functionality

* Reflects calculations based on the User Profile data stored locally.
* Updates daily based on the locally calculated plan.

## Access

* Typically accessible via a main navigation tab or menu item.
